SYSNUTRI

Sistema responsável para auxiliar na elaboração de planos dietéticos conforme a necessidade do paciente. Algumas funcionalidades ainda serão inseridas.

Cadastro de Nutricionistas

Sistema permite cadastro de nutricionistas credenciados com CRN. Painel para acompanhamento de evolução de seus pacientes.

Agendamento de Consultas

O sistema terá uma regra de negócio bem elaborada para marcação de consultas de forma simples e rápida.

Elaboração de Dietas

Elaboração de dietas com diversos alimentos com tabela nutricional baseado no TACO, ou alimentos inseridos por nutricionistas credenciados.

Acompanhamentos dos Pacientes

Acompanhamento dos Paciente: Dietas, Consultas, Relatórios e disponibilização de relatórios para auxiliar os nutricionistas nas tomadas de decisões.
